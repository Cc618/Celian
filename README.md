## Build
Here is a list of commands to test / build / deploy this website.

### Testing (debug mode with hot reload)
```sh
npm install # Do this only once (or when dependencies change)
npm start
```

### Build (release mode)
```sh
npm run build
```

### Github Pages deployment (release mode)
```sh
npm run deploy
```
